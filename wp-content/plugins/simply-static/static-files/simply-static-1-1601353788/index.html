ORIGIN HEALTH PTE. LTD. (SINGAPORE)

Computer Vision and Deep learning Researcher

Project: Image Enhancement of Fetal Ultrasound Images Using Deep Learning

Previous
Next

Origin Health Pte. Ltd. (Singapore) envisions to automate the detection of birth defects from fetal ultrasound scans, a procedure that demands expertise in radiology. There are about a dozen of Ultrasound devices in the market which are used by the doctors. Each device has its own drawbacks and affects images in a different way such as intensity inhomogeneity, contrast loss, etc. Mathematical operators are enough to tackle the problem for a particular device and for a particular artifact, but one cannot generalize over devices. Let us say we have 12 different devices, so for each device, which scans in 20 different standard planes, we require 12 x 20 segmentation networks in total. In order to cope up with this problem, we enhance the image to an extent that irrespective of the device used, all the images will be similar in nature, i.e., reduce the enhanced differences between any two images from different devices and hence, make them indifferentiable. We propose a method by which a DNN can be used to generalize this process of enhancement for any sort of artifact that it carries. In this case finally only one segmentation network is required. I developed a custom deep learning framework using PyTorch as the backend and used it to train neural networks for a range of subtasks, including fetal-tissue Denoising, Harmonization and Enhancement and optimal plane detection for ultrasound scans. My contributions served as a sound foundation for the companys automation motive. This opportunity provided me with a deeper insight into the applications of AI and Computer Vision in medical diagnosis and taught me to work with little and sensitive data. In addition, Confluence was used for documentation and Jira Software was used for Agile Software Development.

Research Assistant

Project: Part-based Segmentation Of 3D CAD Mesh Model Using Deep Learning

Previous
Next

Segmentation literally means division into separate segments, parts or sections. In the case of 3D CAD models as inputs, segmentation results in identifying the parts or components present in the input CAD model. This is to be achieved using a deep learning framework. Owing to the availability of large annotated datasets and also enough computational power in the form of GPUs, many deep learning-based solutions for object classification have been proposed of late, especially in the domain of images and 3D Graphical Models. Nevertheless, very few solutions have been proposed for the task of classifying 3D CAD Models. This is largely due to the lack of a good publicly available dataset suitable for learning-based methods, and also because of the inherent complexities that make it a class unto itself. Also, traditional segmentation algorithms for graphical models have been found to be not working well for low-poly CAD models as an input. Also, algorithms that do work for CAD mesh models require user-intervention / manual parameter setting. My internship revolved around developing novel Deep Learning algorithms for accomplishing automatic part-based segmentation.

MACHINE ASSISTED REHABILITATION OF KNEE OSTEOARTHRITIS (MARKO)

A bio-inspired solution for Rehabilitation of patients suering from Knee Osteoarthritis

Project Objective: Automate both stages of rehabilitation (i.e.)

Stage 1 (THERAPY): To gain control over the knees,
Stage 2 (TRAINING): To strengthen the muscles around the knees,

Integrate them into the same machine and make the recovery process interesting and motivating by interfacing stage 2 with a gaming context. For stage 1, a four-bar linkage slider-crank mechanism assists the leg, performing physiotherapy in a CPM with the help of a linear Motor-driven slider. For stage 2, EMG signals are taken from the patients thigh muscles to detect the intent and direction in which the person puts the effort to move his/her leg and a rotary encoder is used to measure the distance moved by the leg, which is fed as parameters of control for the game, allowing the person to train his leg while gaming.

Previous
Next

Project work summary:

  • Fabricated an efficient design, based on minimum force on the knee joint and cost-effective Continuous passive Motion (CPM) machine.
  • Kinematically synthesized a linkage-based mechanism with the three-accuracy-point approach.
  • Optimized the mechanism to the least amount of power generated by the driver joint and least force generated at the knee joint.
  • Processed Electromyograph signals from Thigh muscles, the Hamstrings and quadriceps and used a Machine Learning algorithm to set the threshold parameter to classify muscle intent signals.
  • Designed a basic game using Blender software, which will be controlled using the EMG and the encoder data. The game deals with the psychological aspects of rehabilitation.

Augmented REALITY (AR) USING ARUCO MARKER & OPENGL

An aruco marker is afiducial markerthat is placed on the object or scene being imaged. It is a binary square with black background and boundaries and a white generated pattern within it that uniquely identifies it. The black boundary helps to make their detection easier. They can be generated in a variety of sizes. The size is chosen based on the object size and the scene, for successful detection. If very small markers are not being detected, just increasing their size can make their detection easier. The idea is that you print these markers and put them in the real world. You can photograph the real world and detect these markers uniquely. After detection, the coordinates of the aruco maker in the frame is recorded in real-time. Now using Blender I had created a black cube which using OpenGL I loaded and positioned at the coordinates of aruco marker.
